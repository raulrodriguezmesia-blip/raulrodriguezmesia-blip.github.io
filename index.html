Hackathon

Agents League Hackathon 2026

Participación en el track Reasoning Agents con Eros Code Analysis Agent, un agente de análisis de código construido para Microsoft Foundry y Azure.

Qué se construyó

Un agente multi-paso que analiza código vulnerable y genera reportes estructurados con hallazgos, recomendaciones y propuestas de mitigación.

  • Pipeline de 5 fases: sintaxis, calidad, seguridad, rendimiento y refactorización.
  • Integración conceptual con Microsoft Foundry IQ y Foundry Toolbox.
  • Salida determinista en JSON para consumo por herramientas o dashboards.
  • Demo técnica, simulación local y tests internos de validación.
  • Enfoque profesional: sin credenciales hardcodeadas, configuración por entorno y documentación técnica.
