# ☕ Java Junior Portfolio - CRUD en Consola

![Java](https://img.shields.io/badge/Java-17+-blue?style=for-the-badge&logo=openjdk)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)

## 📋 Descripción

Proyecto mínimo viable para desarrollador Java Junior. Implementación de CRUD (Create, Read, Update, Delete) en consola con persistencia en archivos. Ideal para aprender los fundamentos de la programación orientada a objetos.

## 🎯 Objetivos de Aprendizaje

- ✅ Entender los principios de POO (Programación Orientada a Objetos)
- ✅ Implementar CRUD básico en Java
- ✅ Persistencia de datos en archivos
- ✅ Manejo de excepciones
- ✅ Estructuras de datos fundamentales

## 🛠️ Tecnologías Utilizadas

- **Lenguaje**: Java 17+
- **IDE**: IntelliJ/Eclipse
- **Persistencia**: Archivos de texto

## 📁 Estructura del Proyecto

```bash
java-junior-portfolio/
├── src/
│   └── main/
│       └── java/
│           └── com/ferreteria/
│               ├── Main.java
│               ├── Persona.java
│               └── Utilidades.java
├── data/
│   └── personas.txt
└── README.md
```

## 🚀 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/raulrodriguezmesia-blip/java-junior-portfolio.git

# Navegar al directorio
cd java-junior-portfolio

# Compilar
javac src/main/java/com/ferreteria/Main.java

# Ejecutar
java -cp src/main/java com.ferreteria.Main
```

## 📊 Funcionalidades

- ✅ Crear nuevas personas/registros
- ✅ Listar todos los registros
- ✅ Actualizar información
- ✅ Eliminar registros
- ✅ Persistencia automática en archivos
