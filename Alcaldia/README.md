# 🏛️ Alcaldia - Sistema de Control de Acceso

![Java](https://img.shields.io/badge/Java-17+-blue?style=for-the-badge&logo=openjdk)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)

## 📋 Descripción

Sistema de control de acceso para pantalla principal con funcionalidades de administración de usuarios y gestión de backups. Aplicación desarrollada con arquitectura modular para asegurar la escalabilidad y mantenibilidad del código.

## 🎯 Funcionalidades

- ✅ Control de acceso basado en roles de usuario
- ✅ Panel de administración para gestión de usuarios
- ✅ Sistema de resumen y generación de backups
- ✅ Interfaz segura para acceso a pantalla principal
- ✅ Arquitectura modular con separación de responsabilidades

## 🛠️ Tecnologías Utilizadas

- **Lenguaje**: Java
- **Framework**: Spring Boot
- **Base de Datos**: MySQL/Oracle
- **Control de Versiones**: Git

## 🚀 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/raulrodriguezmesia-blip/Alcaldia.git

# Navegar al directorio
cd Alcaldia

# Compilar el proyecto
mvn clean install

# Ejecutar la aplicación
mvn spring-boot:run
```
