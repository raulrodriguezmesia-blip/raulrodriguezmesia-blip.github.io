## 📋 Descripción

Aplicación web completa para gestión de inventario de ferretería. Sistema diseñado para manejar productos, stock, ventas y consultas optimizadas con JPA y Hibernate.

## 🎯 Funcionalidades

- ✅ Gestión de inventario con CRUD completo
- ✅ Consultas web optimizadas con JPA
- ✅ Diseño de base de datos relacional
- ✅ Sistema de categorías y productos
- ✅ Registro de ventas y movimientos
- ✅ Búsqueda y filtros avanzados

## 🛠️ Tecnologías Utilizadas

- **Lenguaje**: Java
- **Framework**: Spring Boot
- **ORM**: JPA/Hibernate
- **Base de Datos**: MySQL
- **Frontend**: Thymeleaf/HTML

## 🚀 Instalación

```bash
# Clonar el repositorio
git clone https://github.com/raulrodriguezmesia-blip/ferreteria.git

# Navegar al directorio
cd ferreteria

# Configurar base de datos en application.properties

# Compilar y ejecutar
mvn clean install
mvn spring-boot:run
```
